# Spotlite Hub Memo Template

허브는 위키 변경 기록이 아니라 실제 운영과 실무 진행상황을 보는 현황판입니다.

```markdown
## 운영 메모
- 한줄 요약: 
- 진행 맥락: 
- 실무 판단: 
- 다음 확인: 

## 운영 현황판
- 현재 상태: 
- 최근 추진: 
- 다음 액션: 

## 일시별 추진내용
| 일시 | 추진내용 | 실무 의미 | 연결 증적 | 다음 액션 |
| --- | --- | --- | --- | --- |
| YYYY-MM-DD |  |  | [[Wiki/...]] |  |
```

## 작성 규칙

- `한줄 요약`은 메모만 봐도 지금 어떤 상태인지 알 수 있게 씁니다.
- `진행 맥락`은 왜 진행 중이고 어디까지 왔는지 씁니다.
- `실무 판단`은 무엇을 결정/확인/보류해야 하는지 씁니다.
- `다음 확인`은 오늘 또는 이번주에 실제로 확인할 사람, 고객, 산출물, 리스크를 씁니다.
- `일시별 추진내용`은 위키 편집 이력이 아니라 고객 프로젝트의 실제 추진내용을 적습니다.
