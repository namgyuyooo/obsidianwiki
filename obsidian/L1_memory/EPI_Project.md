# EPI 프로젝트 — L1 Memory Snapshot

## 한줄 요약
EPI 공정/장비 지식 허브. 장비 목록·조건과 MOCVD 레시피를 중심으로 여러 프로젝트(특히 PSK 계열)에서 공통 참조하는 지식 레이어.

## 프로젝트 유형
공정/장비 지식 허브 (납품 프로젝트 아님, PSK 계열 공통 자산)

## 현재 상태 (What's happening now)
- 장비 목록 및 조건 문서 존재 (`Equipment.md`)
- MOCVD 레시피는 Common 네임스페이스로 공유 (`[[Wiki/Common/MOCVD_Recipe]]`)
- Project_Overview.md 없음 — 허브 수준에서만 운영 중

## 핵심 구성 문서
- `Equipment.md`: 장비 목록 및 조건
- `[[Wiki/Common/MOCVD_Recipe]]`: PSK 등 여러 프로젝트에서 공통 사용

## 사용법 (How to use)
- PSK 프로젝트 또는 EPI 관련 기술 검토 시 이 허브와 Common/MOCVD_Recipe 함께 참조
- 장비 조건 확인 필요 시 Equipment.md 드릴다운

## 미해결 이슈 / 확인 필요
- Project_Overview.md 생성 여부 검토 필요
- EPI 관련 추가 지식(공정 파라미터, 이슈 이력)의 위키 편입 여부

## 주의사항 (Gotchas)
- 독립 납품 프로젝트가 아니므로 "완료/진행" 상태 분류 부적절
- MOCVD 레시피는 Common에 있으므로 여기서 중복 편집 금지
