# 사업계획서 프로젝트 — L1 Memory Snapshot

## 한줄 요약
특정 고객 납품 프로젝트가 아니라, 사업계획서 작성을 위한 구조·템플릿·가이드라인 허브. 다른 프로젝트의 제안서 작성 시 참조.

## 프로젝트 유형
내부 템플릿 / 지식 허브 (납품 프로젝트 아님)

## 현재 상태 (What's happening now)
- 사업계획서 전체 목차 템플릿 구축됨
- Executive Summary, 시장분석, 재무예측, 조직관리 등 섹션별 가이드 존재
- 정부 R&D 과제 재활용용 Common 자산 세트 추가됨
- Project_Overview.md 없음 (템플릿 허브 성격이므로 정상)

## 핵심 구성 문서
- Template: 전체 사업계획서 목차
- Executive_Summary: 요약 작성 가이드
- Market_Analysis: 시장 분석 가이드
- Financial_Projections: 재무 예측 가이드
- Writing_Guidelines: 작성 방침과 패턴 요약
- Common/Government_RnD_*: 정부과제용 표현, 역할, 체크리스트, 스타터 템플릿

## 사용법 (How to use)
- 새 사업계획서/제안서 작성 시 이 허브를 먼저 읽고 구조 참조
- Writing_Guidelines에서 표현 패턴 확인
- 정부과제면 Common/Government_RnD_Reusable_Wiki_Hub를 함께 읽고 시작
- 특정 섹션(예: 자금요청, 재무예측)은 해당 가이드 문서 드릴다운

## 미해결 이슈 / 확인 필요
- 각 가이드 문서의 실제 내용 채워진 정도 확인 필요
- 실제 수행한 사업계획서(화승, ZEUS 등)와 연결 링크 보강 필요
- Government_RnD 자산에 더 많은 hwp/hwpx 예시 누적 필요
