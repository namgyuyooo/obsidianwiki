전지 분리막 슬릿롤 외관검사 자동화 POC, 폭 100cm/길이 2000m/고속 120~150m/min 대응 검증 단계.

## 프로젝트 유형
POC / 비전 외관검사 자동화 / 2차 보고 완료, 양산 도입 전 단계

## 현재 상태
- 2025.06.27 진행경과 보고 완료 (pdf/pptx)
- 2차 보고: ROI 축소 + 보조조명으로 113~130fps 달성, 고속 구간 진입 가능성 확인
- 양산 도입 전 1카메라 파일럿 → 5카메라 Full Demo → 주름 검출 PoC → 3개월 도입 로드맵 제시

## 핵심 결정사항
- 검사 대상: 2차전지 분리막 슬릿롤 외관검사 (코팅 후 인라인 + 출하 전 최종 2층)
- 현행 방식: 초기 100m 저속 육안검사, 나머지 1900m 무검사 (120~150m/min)
- 광학계: Allied Vision Alvium 1800 U-319, WD 440mm, FOV 250mm/카메라
