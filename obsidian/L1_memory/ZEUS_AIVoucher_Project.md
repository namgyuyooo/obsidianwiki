# ZEUS AI바우처 — L1 Memory Snapshot

## 한줄 요약
EHM + 이미지 솔루션 병행 적용을 검토한 AI바우처. 사업계획서/중간보고/결과보고 모두 존재. 통합 인프라 vs 분리 인프라 논쟁에서 분리형이 우세. 후속 협업 제안 진행 중.

## 프로젝트 유형
AI바우처 수행 완료 + 후속 확장 설계 단계

## 현재 상태 (What's happening now)
- 2024 AI바우처 수행 완료 (사업계획서, 중간보고, 결과보고서 존재)
- 본더/코터 이물·편심 감지 POC 수행 중 언급
- 2025-02: 후속 협업 제안서(v0.5) 공유됨

## 핵심 결정사항
- EHM + 이미지 솔루션 완전 통합은 보수적 접근 → 분리형 인프라가 더 안정적
- 고객 편의성은 통합형 선호 가능하나, 내부에서는 분리형에 무게

## 핵심 수치 / 파일
- 문서: `2024년 AI바우처_사업계획서_v.4.0.pdf`, `제우스 중간보고 작성중.pdf`, `제우스_(양식)최종(결과)보고서_v2.hwpx`, `제우스xRTM 협업 제안_250228_v0.5.pptx`
- 과제: 본더/코터 이물 + 편심 감지 POC

## 미해결 이슈 / 확인 필요
- 결과보고서 기준 실제 성과와 납품 범위
- 최종 고객안이 통합형인지 분리형인지
- 후속 협업 제안이 실제 계약/추가 POC로 이어졌는지

## 주의사항 (Gotchas)
- 성능 수치보다 "복수 솔루션을 어떤 인프라로 묶을 것인가" 프레임이 핵심 자산
- 통합형/분리형 논쟁은 다른 EHM+이미지 동시 제안 건에서도 재사용 가능한 논리
- 사업계획서 경제적 목표 달성 방안 관련 QA 이력 존재 — 유사 질의 대응 시 참고
