PoC 후속 영업 / 스마트공장 지원사업 제출 준비 / 방문 미팅 전 단계

- 2026-04-27 `#sales_team`에서 담당자 호출, 자료 공유, 미팅 참석자 지정 확인
- 로컬 재구성 보고서 기준 PoC 핵심 수치 정리 완료
- 제출안내문 본문과 고객 과제 정의는 아직 미확정
