# Project Overview

쏘닉스 건은 `IDT 소자 품질 검사 PoC`의 기술 검증 결과를 바탕으로, `제조AI특화 스마트공장 구축사업` 제출 가능성을 함께 검토하는 신규 영업/제안형 프로젝트로 보는 편이 맞다. 2026-04-27 `#sales_team` 스레드에서는 내부 담당 호출, 지원사업 자료 공유, PoC 보고서 공유, 방문 미팅 주체 지정이 하루 안에 이어졌기 때문에, 현재 상태는 단순 기술 리뷰가 아니라 `PoC -> 지원사업형 제안 -> 현장 미팅`으로 이어지는 전환 구간이다.

PoC 보고서 기준 대상 데이터는 총 50장 이미지이며, 정상 30장과 불량 20장으로 구성됐다. 검사 영역은 `IDT`, `Metal`, `Non Metal` 3개로 나뉘며, 현재 파이프라인은 `원본 이미지 -> ROI 크롭 -> 영역 분할 -> SAM3 Text Prompting -> 후처리` 구조다. 성과는 `IDT 13/13`, `Metal 4/5`, `Non Metal 4/5`로 요약되며, 미탐 케이스는 `D_3310`, `D_2324`다. 따라서 기술 메시지는 “특정 영역에서는 강한 가능성을 입증했지만, 양산형 확정안이라기보다 추가 개선과 현장 정의가 필요한 PoC”로 정리하는 편이 안전하다.

이번 프로젝트의 실무 핵심은 두 층이다. 첫째, 고객이 실제로 해결하려는 현장 문제를 PoC 수치와 연결해 `왜 지금 사업으로 묶어야 하는가`를 설명하는 것. 둘째, 공유된 `제조AI특화 스마트공장 구축사업 사업계획서 제출 안내문`과 `제출서류 양식.zip` 기준으로 어떤 제출 트랙인지, 누가 신청 주체인지, 쏘닉스와 RTM의 역할분담이 무엇인지 확정하는 것이다.

## 한 줄 요약

- 쏘닉스 프로젝트는 `IDT 검사 PoC 성과`를 `제조AI특화 스마트공장 구축사업` 제안/미팅 준비로 연결하는 탐색 단계의 신규 프로젝트다.

## 프로젝트 성격

- 유형: PoC 후속 영업 / 정부지원사업 제출 준비 / 스마트공장 제안형 프로젝트
- 고객: 쏘닉스
- 현재 단계: 미팅 전 초안 정리 및 사업기회/제출요건 확인 단계
- 내부 관련자:
  - 허정: 최초 스레드 생성 및 자료 공유
  - 유남규: 미팅 참석 주체
  - 조서연: 과제 유형 질의 및 후속 공유 수신

## 핵심 판단

- 이 건은 단순 “PoC 결과 전달”이 아니라 지원사업 제출 가능성까지 엮인 영업기회다.
- 제안 메시지는 `IDT 영역 전건 탐지`를 전면에 두되, `Metal/Non Metal 미탐`과 `추가 샘플/threshold 조정 필요`를 숨기지 않는 편이 이후 신뢰에 유리하다.
- 제출안내문 본문을 아직 직접 읽지 못했으므로, 사업명/주관기관/평가 기준/일정은 확정 사실로 쓰지 말고 미팅 확인 항목으로 둬야 한다.
