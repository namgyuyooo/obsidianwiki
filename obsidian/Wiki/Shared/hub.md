# Shared Hub

여러 프로젝트에서 반복 활용하는 고객 유형, 사업화 전략, 문장 자산을 모아두는 영역입니다.

- 향후 공통 고객 세그먼트, 사업화 전략, 재사용 문장 자산을 이 네임스페이스에 승격합니다.
- 프로젝트 문서에는 Shared 문서를 직접 링크하고, 프로젝트 고유 문맥만 남기는 것을 원칙으로 합니다.
- 공통 승격 작업 시에는 [[Wiki/Common/Wiki_Ingest_Prompt_Set]]의 `공통 지식 승격` 프롬프트를 우선 사용합니다.
