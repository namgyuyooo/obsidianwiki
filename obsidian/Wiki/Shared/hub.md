# Shared Hub

- [[Wiki/index]]

## 운영 메모
- 한줄 요약: 프로젝트에서 승격된 재사용 자산을 실무 기준으로 관리하는 허브입니다.
- 진행 맥락: 이 공간은 개별 프로젝트 실행보다 여러 프로젝트에 반복 적용되는 규칙과 자산을 다룹니다.
- 실무 판단: 지금 실제로 쓰이는 문서, 보강이 필요한 운영 자산, 승격 후보를 계속 가시화해야 합니다.
- 다음 확인: 사용 빈도가 높은 문서, 비어 있는 자산, 새로 승격할 후보를 정리합니다.

## 재사용 자산
- 공통 승격 작업 시에는 [[Wiki/Common/Wiki_Ingest_Prompt_Set]]의 `공통 지식 승격` 프롬프트를 우선 사용합니다.

## 승격 / 정비 큐
- 보강 필요 자산: 현행 프로젝트 운영에 직접 쓰이는 문서부터 우선 보강
- 프로젝트에서 승격할 후보: 반복되는 판단 로직, 문장 자산, 운영 패턴 분리
- 재검토 필요 문서: 오래된 규칙, 중복 문서, 링크만 있고 실무 가치가 약한 문서 점검

## 자동화 / 거버넌스 진입점
- 관련 운영 문서와 자동화 진입점 추가 필요

## 핵심 링크
- 공통 승격 작업 시에는 [[Wiki/Common/Wiki_Ingest_Prompt_Set]]의 `공통 지식 승격` 프롬프트를 우선 사용합니다.

## 기존 연결 메모
여러 프로젝트에서 반복 활용하는 고객 유형, 사업화 전략, 문장 자산을 모아두는 영역입니다.

- 향후 공통 고객 세그먼트, 사업화 전략, 재사용 문장 자산을 이 네임스페이스에 승격합니다.
- 프로젝트 문서에는 Shared 문서를 직접 링크하고, 프로젝트 고유 문맥만 남기는 것을 원칙으로 합니다.
- 공통 승격 작업 시에는 [[Wiki/Common/Wiki_Ingest_Prompt_Set]]의 `공통 지식 승격` 프롬프트를 우선 사용합니다.
