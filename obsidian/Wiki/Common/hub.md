# Common Hub

- [[Wiki/index]]

## 운영 메모
- 한줄 요약: 위키 운영 모델, 자동화, 거버넌스를 실무 기준으로 관리하는 허브입니다.
- 진행 맥락: 이 공간은 개별 프로젝트 실행보다 여러 프로젝트에 반복 적용되는 규칙과 자산을 다룹니다.
- 실무 판단: 지금 실제로 쓰이는 문서, 보강이 필요한 운영 자산, 승격 후보를 계속 가시화해야 합니다.
- 다음 확인: 사용 빈도가 높은 문서, 비어 있는 자산, 새로 승격할 후보를 정리합니다.

## 승격 / 정비 큐
- 보강 필요 자산: 현행 프로젝트 운영에 직접 쓰이는 문서부터 우선 보강
- 프로젝트에서 승격할 후보: 반복되는 판단 로직, 문장 자산, 운영 패턴 분리
- 재검토 필요 문서: 오래된 규칙, 중복 문서, 링크만 있고 실무 가치가 약한 문서 점검
