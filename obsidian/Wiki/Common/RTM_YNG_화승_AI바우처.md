# 화승 AI바우처

## Project Overview

- 과제명은 `믹싱 데이터 기반 자동차용 복합형상 웨더스트립 연속 제조공정 최적화 및 물성 예측 솔루션`이다.
- 주관 수요기업은 `㈜화승코퍼레이션`, 참여 공급기업은 `㈜알티엠`으로 확인된다.
- 수행기간은 `2025.05.01 ~ 2025.11.30`이며, AI바우처 일반분과 사업계획서 성격의 대표본이다.
- 도입 솔루션명은 `EHM`이며, 설비 이상 탐지와 가상계측 기반 물성 예측을 동시에 추진하는 제조 데이터 프로젝트로 읽힌다.

## Technical Scope

- 주요 센서 데이터와 공정 로그를 기반으로 설비 상태를 실시간 분석하고 이상 징후를 조기 탐지한다.
- 가상계측 모델을 통해 배치별 물성치와 품질 변동을 예측한다.
- 공정 최적화 시스템을 통해 조건 추천과 제조 효율 향상을 목표로 한다.
- 대표 기능은 `Pattern Scoring`, `Step Analysis`, `Representative Value Detection`, `Time-Series Pattern Recognition`으로 정리된다.

## Business Value

- 샘플링 중심 검사 한계를 줄이고 데이터 기반 전수 예측으로 전환하려는 과제다.
- 설비 이상 조기 인지와 물성 예측을 통해 불량률 감소, 다운타임 최소화, 품질 안정성 확보를 동시에 노린다.
- 화승 글로벌 네트워크와 자동차 부품 공급망을 활용해 해외 확장과 유사 공정 확장 가능성도 사업계획서에 포함되어 있다.

## Related Workstreams

- Drive 발견 기준으로 `1.POC / 화승` 폴더에는 `HS_촬영본.zip`, `frames_15fps`, `mp4`, `json`, `png` 결과물이 별도로 존재한다.
- 따라서 이 프로젝트는 사업계획서 트랙과 영상/이미지 기반 PoC 트랙이 병행되었을 가능성이 높다.
- 현재 읽힌 대표본은 사업계획서이므로, 운영 위키 기준 정보는 본 문서를 우선 사용하고 시각 증거 세트는 후속 보강 대상으로 둔다.

## Version Notes

- 현재 확인된 파일명만으로도 `0318 -> 0321 -> 0322 -> 0323 -> 인건비 삭제본` 순서가 선명하다.
- `0318`은 `hwp`, 이후는 `hwpx`로 이어져 작성 도구/편집 경로가 바뀐 흔적이 보인다.
- `0322`와 `0323`은 핵심 기관정보, 사업비 소요명세, 공급기업 인원 표기가 거의 같아 실제로는 같은 정리 단계의 연속본으로 보인다.
- 반면 최종 `인건비 삭제본`에서는 요약 사업비 숫자 `230,000 / 12,000 / 188,000 / 200,000 / 430,000`이 전면에 채워지고, 공급기업 인원 수치가 `36명 -> 35명`으로 한 번 더 조정된다.
- `인건비 삭제` 이후에는 `0409`, `0410` 발표자료가 이어져, 문서 흐름이 `계획서 제출 -> 발표평가 대응`으로 넘어간다.
- 이 계보는 단일 최종본만 있는 것이 아니라 실무 수정 이력이 명확히 남은 사례라서 Change Log 대상 가치가 높다.
