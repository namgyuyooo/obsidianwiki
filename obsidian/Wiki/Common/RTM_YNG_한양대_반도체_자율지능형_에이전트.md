# 한양대 / 반도체 자율지능형 에이전트

## Project Overview

- 대표본은 `2026년 인공지능 기술사업화 지원사업 연구개발계획서_통합_0310.pdf`이며, 과제번호는 `CY260116`이다.
- 과제명은 `반도체 장비 운용최적화를 위한 산업 특화 대형언어모델 기반 자율지능형 에이전트 개발`이다.
- 수행기간은 `2026.04.01 ~ 2026.12.31`의 `9개월`이며, 반도체 유지보수 엔지니어를 위한 `On-Premise` 기반 의사결정 지원 시스템 실증이 핵심 목표다.

| 기관 | 역할 |

| 데이터/RAG 관리 | 데이터 소스 관리, 버전 갱신, 이동식 저장매체 기반 업데이트 |

## Knowledge Points

- `10만건 이상`의 도메인 문서를 참조하는 `PE Agent` 구상이 직접 명시된다.
- 핵심 구조는 `문서 구조화`, `지식 그래프`, `검색 모델`, `Multi-Agent 추론`의 4축 결합이다.
- 폐쇄망 `Fab` 환경을 전제로, `이동식 저장매체 기반 RAG 업데이트` 표준 절차를 강조한다.
- `Graph-aware RAG`와 의미 동일성 판별을 통해 오답 통제와 답변 일관성을 확보하려는 점이 특징이다.

## Distinction From Other Semiconductor Agent Tracks

- [[RTM_YNG_반도체_공정제어_멀티에이전트]]는 `RS-2026-25523818` 기반 장기형 공정제어 과제다.
- 이번 문서는 `CY260116` 기반 `9개월` 기술사업화 과제로, 초점이 `유지보수 의사결정 지원`, `On-Prem LLM`, `Fab 실증`에 더 강하게 놓여 있다.
- 즉, 둘 다 반도체 에이전트 축이지만 사업명, 과제번호, 기간, 참여기관 구성, 상용화 방식이 달라 별도 프로젝트로 관리하는 편이 맞다.

## Version Notes

- `기타업무 / 대학연구소-스타트업 공동 혁신 R&D 제안서` 폴더에서 `발췌본`, `취합본`, `스타일포함본`, `부속 본문`, `최종 발표본`이 한 흐름으로 확인된다.
- 즉, 현재 `한양대` 프로젝트는 단일 `pdf`만 있는 것이 아니라 `250124 -> 250218 -> 250219 -> 250224 -> 0310 통합본`으로 이어지는 조상 문서 계보를 가진다.
- `매출액수정` 표기와 `기업현황`, `성능지표` 파일 존재로 볼 때, 이 과제는 기술 본문뿐 아니라 사업화 수치와 부속표를 별도 작업한 흔적이 뚜렷하다.
- 발표본 제목이 `반도체 Fab 무인화를 위한 기반 기술 개발 및 상용화`로 남아 있어, 제출본과 발표본 사이에서 외부 설명용 메시지가 따로 정리됐을 가능성이 높다.

## Risks

- 현재 읽힌 것은 통합 `pdf` 대표본이며, `기타업무` 쪽 조상 `hwp/hwpx` 계보는 파일명 기반으로만 연결했다.
- 즉, 버전 사슬은 확인됐지만 각 중간본의 본문 diff와 수치 변화는 아직 정밀 비교하지 못했다.
- 사업화 수치와 기술 성능 목표는 본문에서 읽히지만, 실제 최종 제출본에서 조정됐을 가능성은 남아 있다.
