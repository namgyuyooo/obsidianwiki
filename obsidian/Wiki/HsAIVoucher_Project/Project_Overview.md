# Project Overview

화승 AI바우처 프로젝트는 고무 물성예측 솔루션을 중심으로 한 바우처형 사업 프로젝트로 보인다. 공개 Slack 기준으로 사업계획서 최종본, 최종평가 결과 통보, 검수확인서, 기능추가개발 견적서가 연결되며, 2026년 4월에는 설비 수량 정정에 따른 견적 수정 재발송이 확인된다.

## 한 줄 요약

- 화승 프로젝트는 바우처 사업 완료 이후 기능추가개발 견적까지 이어지는 확장형 프로젝트다.

## 프로젝트 성격

- 유형: AI바우처 기반 사업 + 후속 기능추가개발
- 고객: 화승코퍼레이션
- 주요 키워드:
  - 고무 물성예측
  - AI바우처
  - 검수
  - 최종평가
  - 기능추가개발 견적

## 현재까지 확인된 실제 정보

- 사업 단계:
  - 사업계획서 최종본 존재
  - 최종평가 결과 통보 존재
  - 검수확인서 존재
- 확장 단계:
  - 기능추가개발 견적서 존재
  - 설비 수량 정정 후 견적서 재발송 메시지 존재
  - 2026-04-02 기준 전체 양산공장 `5개 설비` 확산전개 문의가 들어왔으나, 실제 확장 견적 기준은 `1~4호기 추가`로 재정렬된다.
  - 2026-04-08 메시지에서는 `이전 기능추가개발 견적`과 `확산 전개 견적`이 별개라고 명시된다.
- 상태 해석:
  - 바우처 사업 자체는 일정 수준 마감되었고, 이후 고객 확장 요구를 반영한 후속 개발 단계로 넘어가는 흐름이다.

## 핵심 엔티티

- 고객:
  - 화승코퍼레이션
- 주제:
  - 고무 물성예측 솔루션
- 문서:
  - 사업계획서 최종본
  - 최종평가 결과
  - 검수확인서
  - 기능추가개발 견적서

## 현재 상태

- 기존 바우처 사업 결과물은 검수 및 평가 문서로 확인된다
- 2026년 3~4월에는 설비 확장과 기능추가개발 견적 논의가 진행된다
- 따라서 현재 기준 이 프로젝트는 종료형이 아니라 후속 확장형 프로젝트에 가깝다
- 후속 확장은 `기존 솔루션 복제/전개 비용`과 `신규 기능개발 비용`을 분리해 산정하는 방향으로 보인다

## 위키에 남겨야 할 포인트

- 화승은 `사업 완료 증적`과 `후속 확장 견적`이 함께 있는 사례다.
- 설비 대수 오인과 수정 재발송 이력은 수량/범위 관리의 대표 사례로 남길 가치가 높다.
- 같은 고객의 “운영 확산”과 “추가 기능개발”이 반드시 같은 견적 항목은 아니라는 점이 중요하다.

## 확인 필요

- 기존 사업 범위와 기능추가개발 범위가 어떻게 달라지는지
- 4대/5대 설비 기준이 최종적으로 어떻게 확정됐는지
