## 기존 정리 메모
서울바이오시스 프로젝트는 EHM 데모 POC와 AI agent 통합 인사이트를 축으로 전개되는 검증-확장형 프로젝트로 보인다. 공개 Slack 기준으로 2025년에는 EHM 소개 및 데모 POC 결과 공유, 2차 결과 보고 문서가 남아 있고, 2026년 2월에는 AI agent 통합 인사이트 자료가 이어진다.

## 한 줄 요약

- 서울바이오시스는 EHM 데모 POC 결과를 바탕으로 AI agent 방향까지 확장 검토하는 프로젝트다.

## 프로젝트 성격

- 유형: 데모 POC 기반 확장형 프로젝트
- 고객: 서울바이오시스
- 주요 키워드:
  - EHM
  - 데모 POC
  - 결과 공유
  - AI agent 통합 인사이트

## 현재까지 확인된 실제 정보

- 2025년 단계:
  - EHM 소개 및 데모 POC 결과 공유 문서 존재
  - EHM 데모 POC 2차 결과 보고 문서 존재
- 2026년 단계:
  - AI agent 통합 인사이트 발표자료 존재
  - 최종보고 자료 크로스체크 요청 메시지 존재
- 후속 논의:
  - 안산과 VINA 데모 결과는 전반적으로 긍정적으로 보인다는 메시지가 확인된다.
  - 연휴 이후 대표 대상 최종보고와 next step 논의가 예정된 것으로 보인다.
  - VINA 현장 데모를 통해 기존 Synopsys 대비 EHM 강점에 대한 공감대가 형성됐고, `3건의 개선사항`, `2건의 신규 개발 요청사항`이 도출됐다는 메시지가 있다.
  - 후속 논의에서는 도입 검토 주체가 안산 본사에서 VINA 측으로 이동했다는 설명이 나온다.
  - 가격 논의에서는 `123대 기준 22억 -> 19억` 제안이 있었지만 여전히 비싸다는 반응이 남아 있다.
  - 최종보고 준비 문맥에서는 `FDC 기능`, `AI 고도화 기능`, `HW 시스템 구축`, `유지보수`를 분리해 설명하려는 구조가 보인다.
  - 2025-11-11 메시지에서는 의사결정이 지연되고 있으며 IT 부사장 대상 추가 소개와 데모 결과 공유가 필요하다고 정리된다.
- 상태 해석:
  - 초기 데모 POC에서 끝난 것이 아니라, 결과를 바탕으로 다음 단계 솔루션 방향까지 논의한 것으로 보인다.
  - 다만 긍정적 검증이 곧바로 계약으로 이어진 것은 아니고, 가격과 의사결정 구조가 주요 병목으로 남아 있다.

## 현재 상태

- 데모 POC 결과는 최소 2차 보고까지 누적된 상태다
- 이후 AI agent 관점의 통합 인사이트 자료가 추가되어 확장 검토 단계로 해석된다
- 공개 메시지 기준으로는 POC 결과가 부정적으로 종료된 것이 아니라, 최종보고와 next step 논의까지 이어지는 긍정적 검증 흐름으로 보인다
- 그러나 실제 도입 단계에서는 가격 조정과 본사/VINA/IT 의사결정 라인 정리가 필요해 전환이 늦어지는 상태다

## 위키에 남겨야 할 포인트

- 서울바이오시스는 `POC 결과 축적 -> AI agent 확장 논의` 구조가 핵심이다.
- EHM 단일 솔루션 소개에서 더 넓은 agent 방향으로 이동한 흔적을 추적할 가치가 높다.
- `안산`과 `VINA` 두 현장/법인 단위 결과가 함께 언급된다는 점도 별도 관리 포인트다.
- 긍정적 피드백과 실제 도입 결정 사이에 `가격`, `기존 FDC 대비 설명`, `의사결정 주체 이동`이 병목으로 작동한다는 점이 중요하다.

## 확인 필요

- POC 결과에서 어떤 지표나 인사이트가 나왔는지
- AI agent 통합 인사이트가 후속 과제 제안으로 이어졌는지
- 최종보고 이후 실제 제안 또는 실행 단계로 전환됐는지
