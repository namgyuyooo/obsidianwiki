## 기존 정리 메모
한국알박 프로젝트는 ULVAC Sputtering 설비를 대상으로 한 예지보전·공정제어 PoC 제안 프로젝트로 보인다. 공개 Slack 기준으로 `EHM 데모 추진계획 -> 최종 리뷰 -> 사업제안서 구성 작업 -> 견적서 -> PoC 제안서 v2` 흐름이 확인된다.

## 한 줄 요약

- 한국알박 프로젝트는 데모와 제안서를 기반으로 PoC 수주를 준비하는 제안-전환형 프로젝트다.

## 프로젝트 성격

- 유형: PoC 제안형 프로젝트
- 고객: 한국알박
- 주요 키워드:
  - ULVAC Sputtering
  - 예지보전
  - 공정제어
  - EHM 데모
  - PoC 제안서

## 현재까지 확인된 실제 정보

- 제안 흐름:
  - EHM 데모 추진계획서 존재
  - 최종 리뷰 미팅 문서 존재
  - 사업제안서 작업 리스트 존재
  - PoC 프로젝트 견적서 존재
  - PoC 제안서 v2 존재
- 제안 내용 단서:
  - 내부 키워드 리스트에는 `박막 두께 및 Uniformity 최적화`, `Sheet Resistance / Film Stress 예측`, `Target Erosion Profile 예측`, `Target 수명 및 교체 시점 예측`, `Chamber Cleaning 주기 예측`이 포함된다.
  - v2 검토 과정에서는 `Total Life`와 `Target Life`를 혼동하지 말아야 한다는 피드백이 직접 언급된다.
- 상태 해석:
  - 단순 아이디어 단계가 아니라 제안 문서와 가격 문서를 갖춘 수주 준비 단계로 보인다.

## 현재 상태

- 데모 단계에서 출발해
- 제안서 구성과 리뷰를 거쳐
- 견적과 v2 제안서까지 도달한 상태다
- 리뷰 이후에는 표현과 개념 정의를 더 정교하게 다듬는 단계로 들어간 것으로 보인다

## 위키에 남겨야 할 포인트

- 한국알박은 제안 문서 버전 변화가 핵심이다.
- 데모 계획과 최종 제안서 사이에 논리 구조가 어떻게 바뀌는지 추적 가치가 높다.
- 예지보전/공정제어 계열 제안 템플릿의 기준 사례로 활용 가능하다.
- 특히 설비 물리 개념과 수명 개념을 고객 언어에 맞게 정리하는 과정이 제안 성공의 핵심 포인트로 보인다.
