# Field Transition

금호타이어 첫제품 / X-ray 프로젝트의 핵심은 모델 개발보다 `POC -> 현장 기준 -> 1차 납품` 전환이다.

## 단계별 흐름

- 4월 초에는 경과보고 자료, 사용자 매뉴얼, 산출물 목록이 정리되며 설명/전달 문서가 준비된다.
- 4월 10일에는 라이브 화면 수정과 Engine 활용 방향 유지가 결정된다.
- 4월 14일에는 현장 점검, 고객 화상회의, 데이터 구조 확인 액션 아이템이 이어진다.
- 4월 20일에는 데모 피드백을 충분히 수집한 뒤 실제 솔루션에 반영/설치하기로 하며 하드웨어 납품 계획이 잡힌다.
- 이후 작업자 피드백이 반영되면서 확대/축소, 단일 이미지 확대, Mold 번호 검증 같은 현장 요구가 구체화된다.

## 전환 과정에서 바뀌는 것

- POC 단계의 “이 정도 검출이면 충분”이라는 기준이 현장 단계에서는 더 엄격한 검사 기준으로 바뀐다.
- 화면 구성도 연구/시연 기준이 아니라 작업자 가독성 기준으로 재설계된다.
- 검사 항목 자체도 고정되지 않고, 일부 항목 제외나 번호 일치 검증처럼 실제 운영 관점 요구가 추가된다.

## 위키 해석

- 이 프로젝트는 AI 성능 평가 프로젝트라기보다 `업무 기준 정의 프로젝트`에 가깝다.
- 따라서 현장 전환형 프로젝트에서는 `모델 성능`, `검사 기준`, `작업자 화면`, `데이터 테이블`, `설치 일정`이 모두 같은 중요도로 관리돼야 한다.
